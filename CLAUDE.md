## 가장 먼저 읽을 것
- **[OS.md](./OS.md)** — 이 OS의 목표·원칙·설계의 단일 출처(single source of truth).
  모든 작업은 OS.md의 목표/원칙에 부합해야 합니다. 충돌 시 OS.md를 우선합니다.

## 핵심 규칙 (OS.md 요약 — 자세한 건 OS.md 참조)
- **Claude는 자동매매를 하지 않습니다.** 매매 *추천과 근거*까지만. 실제 주문은 항상 사용자가 승인/실행 (Human-in-the-loop).
- **내 실제 보유가 기준** — 보유·매매 기록은 `portfolio/` 원장에 직접 남깁니다.
- **근거 없는 추천 금지** — 데이터·이유·리스크를 항상 함께.
- **계좌 자격증명은 절대 깃/평문 저장 금지** (`.env`/시크릿만).
- **교육 우선** — "종목 찍어주기"보다 "읽는 법"을 가르칩니다.
