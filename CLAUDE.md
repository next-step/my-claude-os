# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a step-by-step mission repository for "나만의 클로드 코드 OS 만들기" (Building My Own Claude Code OS). Each week introduces new requirements that are implemented, submitted via Pull Request, reviewed, and merged before moving to the next step.

## Git Workflow

- Work happens on a personal branch (e.g., `step1`, `minnseong`)
- Submit completed work as a Pull Request to the reviewer's branch
- Address review feedback, push again
- Merge unlocks the next step

## Repository State

The project is in early stages — no build system, test runner, or application code exists yet. As each mission step adds code, update this file with relevant build/test/run commands.
