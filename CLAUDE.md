# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 작업 규칙

1. **클로드 OS 관련 모든 파일은 반드시 프로젝트 안에 만든다.**
   `.claude/` 하위의 md 파일을 포함해 클로드 OS와 관련된 모든 산출물은 이 프로젝트 디렉터리 내부에 생성한다. 사용자 홈 디렉터리(`~/.claude` 등) 등 프로젝트 바깥에는 만들지 않는다.

2. **대화 과정에서 양질의 설명을 제공한다.**
   이 저장소는 "클로드 OS 만들기" 실습 프로젝트다. 사용자가 AI와의 협업 방식을 배울 수 있도록, 무엇을·왜·어떻게 하는지 충분히 설명하며 진행한다.
