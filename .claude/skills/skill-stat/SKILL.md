# Skill Stat 스킬

스킬 호출 통계를 프로젝트별로 읽어 보기 좋게 출력한다.

## 데이터 소스

`~/.claude/skill-usage.json` 파일 하나만 읽는다.

```bash
cat ~/.claude/skill-usage.json 2>/dev/null
```

파일 구조:
```json
{
  "projects": {
    "my-claude-code-os": {
      "skills": { "github-commit": 3, "skill-stat": 1 },
      "last_updated": "2026-06-25T11:35:00Z"
    },
    "other-project": {
      "skills": { "pdf": 2 },
      "last_updated": "2026-06-25T11:00:00Z"
    }
  },
  "last_updated": "2026-06-25T11:35:00Z"
}
```

## 출력 형식

파일의 `projects` 키를 순회하며 프로젝트마다 아래 블록을 출력한다.  
프로젝트가 없거나 모든 skills가 비어 있으면 "아직 기록된 스킬 사용이 없습니다." 출력.

---

### 예시 출력

```
📊 스킬 사용 통계 (마지막 업데이트: 2026-06-25T11:35:00Z)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📂 my-claude-code-os  (총 4회)
마지막 사용: 2026-06-25T11:35:00Z

순위  스킬명           횟수   비율     바
──────────────────────────────────────────────
  1   github-commit    3회  75.0%  ████████████████████
  2   skill-stat       1회  25.0%  ███████░░░░░░░░░░░░░

📂 other-project  (총 2회)
마지막 사용: 2026-06-25T11:00:00Z

순위  스킬명   횟수    비율     바
──────────────────────────────────────
  1   pdf      2회  100.0%  ████████████████████
```

---

## 출력 규칙

- **프로젝트 정렬**: 마지막 사용 시각 내림차순 (최근 사용 프로젝트 먼저)
- **스킬 정렬**: 횟수 내림차순, 동률이면 알파벳 오름차순
- **바 차트**: 해당 프로젝트 내 최대값 기준 20칸 스케일
  - `바 길이 = round(20 × 해당_횟수 / 최대_횟수)`
  - `채운_부분 = "█" × 바 길이`, `빈_부분 = "░" × (20 - 바 길이)`
- **비율**: 해당 프로젝트 총 호출 수 대비 백분율
- **스킬명 열 너비**: 해당 프로젝트에서 가장 긴 스킬명 기준 맞춤
