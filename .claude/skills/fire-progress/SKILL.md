# 📈 파이어 진척도 — fire-progress

파이어 목표까지 **지금 어디쯤 왔고, 이 페이스로 목표 시점에 닿는지**를 보여준다.

> ⚠️ AI는 계산·설명 도우미일 뿐, 투자 권유가 아니다. 예측은 가정(수익률·저축)에 따른 추정일 뿐 보장이 아니다.

## 계산은 스크립트가 (산수를 LLM이 하지 않는다)

```bash
python3 .claude/skills/fire-progress/scripts/progress.py
```

스크립트가 결정론적으로 처리한다:
- **순자산** = 보유수량([portfolio.private.md](../../../portfolio.private.md)) × 라이브 현재가(야후) + 예수금
- **목표금액 · 연 저축 · 연 수익률 가정 · 목표 시점** ← [portfolio.md](../../../portfolio.md)의 "파이어 목표" 섹션
- **진척률** = 순자산 ÷ 목표금액
- **예측** = 순자산 복리 + 매년 저축(연금)을 수익률 시나리오별로 목표 시점까지 굴린 값 → 목표 대비/도달 여부

## 할 일
1. 위 스크립트를 실행한다.
2. 결과 표를 그대로 보여준 뒤 사람 말로 해설한다:
   - 지금 진척률, 목표까지 남은 금액·기간.
   - 시나리오(보수/기본/낙관)별로 목표에 닿는지, 갭이 크면 얼마나 부족한지.
3. 갭이 있으면 **선택지**를 사실로 제시한다(권유 아님): 저축을 늘리거나 / 목표금액을 조정하거나 / 기간을 늘리면 숫자가 어떻게 변하는지. (가정값은 portfolio.md에서 바꾸면 즉시 재계산됨)
4. 조회 실패 종목이 있으면 사용자에게 현재가를 물어 보완한다.

## 규칙
- 결과는 화면에만. 저장하지 않는다(매일 변하는 파생값).
- **가정을 분명히 밝힌다.** "수익률 X% 가정" 없이 예측을 단정하지 않는다.
- 투자 권유가 아니라 사실·계산 전달. 격려는 하되 "그러니 뭘 사라"는 하지 않는다.
