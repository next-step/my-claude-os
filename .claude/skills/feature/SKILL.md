# /feature — 기능 개발 흐름 안내/런처

[OS.md](../../../OS.md) 개발 흐름의 **진입점 겸 안내자**다. 이 스킬은 **직접 일을 하지 않는다.**
PRD·Plan 작성은 각 단계 스킬(`/prd`, `/plan`)이 하고, 흐름 강제는 CLAUDE.md 규칙이 한다.
`/feature`는 그 사이에서 **"지금 어디고, 다음이 뭔지"를 알려주고 다음 단계 스킬을 실행**한다.

> 왜 런처로 두나: 단계가 늘어도(③ 구현 등) 사람은 `/feature` 하나만 기억하면 된다.
> 그리고 단계 사이에 "상태 파악·안내"라는 가벼운 연결 고리만 맡아 **얇게 유지**한다.
> (단계만 콕 집어 하고 싶으면 `/prd`·`/plan`을 직접 써도 된다.)

전달된 인자: ``
- 인자가 있으면 → 기능 설명/slug로 사용한다.
- 인자가 없으면 → "어떤 기능을 진행할까요?"라고 한 줄로 묻고 시작한다.

이 스킬의 범위는 **①~③ 흐름 안내**다(각 단계 스킬을 상태에 맞게 실행). ④ 마무리(`/commit`·`/push`)는 사람이 결과를 확인한 뒤 진행한다.

---

## 진행 순서

### 1. slug 결정 + 현재 진행 상태 스캔

기능 설명에서 slug를 정한 뒤, 어느 단계까지 왔는지 산출물 존재로 파악한다.

```bash
test -f docs/requirements/<slug>.md && echo REQ
test -f docs/prd/<slug>.md && echo PRD
test -f docs/plan/<slug>.md && echo PLAN
```

> 파일 존재는 "그 단계 산출물이 만들어졌다"는 신호일 뿐, "확정"까지는 보장하지 않는다.
> 확정 여부는 각 단계 스킬의 게이트에서 사람이 정한다. 런처는 거기에 관여하지 않는다.

### 2. 흐름 지도 보여주기

현재 상태를 ✅(완료)/▶(다음)/⬜(대기)로 표시해, 사람이 큰 그림을 보게 한다.

```
🗺️ <기능 이름> 개발 흐름

① 요구사항·PRD   [✅ / ▶ / ⬜]   → /prd
② Plan(설계)      [✅ / ▶ / ⬜]   → /plan
③ 구현            [✅ / ▶ / ⬜]   → /impl (구현↔검증 루프)
④ 마무리          [⬜]            /commit · /push
(게이트=사람 검토는 ①·② 각 단계 끝에 내장 — 별도 번호 없음)
```

### 3. 다음 단계 스킬 실행

상태에 따라 다음 한 걸음만 진행한다. **`Skill` 도구로 해당 단계 스킬을 호출**한다.

| 현재 상태 | 다음 동작 |
|-----------|-----------|
| PRD 없음 | `/prd` 실행 (요구사항→PRD) |
| PRD 있음, Plan 없음 | `/plan` 실행 (PRD→설계) |
| PRD·Plan 둘 다 있음 | `/impl` 실행 (구현↔검증 루프) — Plan이 승인된 상태인지 먼저 확인 |

- 호출한 단계 스킬은 **자체 게이트에서 멈춘다.** 그 멈춤을 존중한다 —
  런처가 게이트를 넘겨 다음 단계로 밀어붙이지 않는다.
- 한 번에 한 단계만 진행하고, 단계가 끝나면 다시 이 지도를 보여줘 다음을 안내한다.

---

## 수용 기준 (이 스킬이 제대로 동작하는지 검증)

- [ ] `/feature "샘플"` 실행 시 산출물 존재를 스캔해 **현재 단계를 정확히 판별**한다.
- [ ] 흐름 지도를 ✅/▶/⬜로 보여준다.
- [ ] PRD가 없으면 `/prd`를, PRD만 있으면 `/plan`을 **`Skill` 도구로 실행**한다.
- [ ] 런처는 **직접 PRD·Plan을 작성하지 않는다**(Write·Task를 쓰지 않는다).
- [ ] 호출한 단계 스킬의 게이트에서 멈추고, 임의로 다음 단계로 밀어붙이지 않는다.

---

## 안전 규칙

- **런처는 일을 위임할 뿐, 직접 하지 않는다.** 문서 작성·코드 수정은 단계 스킬의 몫이다.
- **게이트를 넘지 않는다.** 각 단계 스킬이 멈추면 그 멈춤을 존중한다.
- 한 번에 한 단계만. 흐름 순서를 건너뛰지 않는다(PRD 없이 Plan으로 가지 않는다).
