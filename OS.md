# 📈 내 장기 포폴 시장 파악 OS

## 목표

지금 나는 사실상 아무것도 모르는 상태로 장기투자를 하고 있다.
이 OS로 **내가 가진 장기 포트폴리오를 중심으로, 시장과 세계가 어떻게 돌아가는지 알아가는 것**이 목표다.
내 보유 종목·섹터에 무슨 일이 벌어지는지 매일 파악하면서,
필요할 때 **리밸런싱을 어떻게 하면 좋을지** 스스로 판단할 수 있는 감을 키운다.

> ⚠️ AI는 분석·설명을 해주는 도우미일 뿐, 따라야 할 투자 권유가 아니다.
> 판단과 실제 매수/매도 실행은 내가 한다.

## 설계 구상

**누가 뭘 하나**
- 나(사람): 내 포폴 공유, 궁금한 것 던지기, 판단·실제 매매 실행
- AI(도우미): 내 보유 종목·섹터 관련 시장/뉴스 브리핑, 배경 설명, 리밸런싱 시나리오 제안, **그날 대화 정리 + 복습 퀴즈 생성**

**어떻게 도나 (반복 루프)**
```
1. [AI]  내 보유 종목·섹터 관련해서 시장이 어떻게 돌아가는지 브리핑
2. [나]  읽고 궁금한 것 물어보기 / 내 생각 메모
3. [AI]  (필요 시) 비중·리스크 진단 + 리밸런싱 시나리오 제안
4. [나]  판단 → 실제 매매는 직접
5. [AI]  그날 대화를 노트로 정리하고, 복습 퀴즈를 만들어 남김  ← 기록·체득
6. ↻    다음 날 또 반복 (리밸런싱은 가끔)
```

**데이터**
- 내 장기 포트폴리오 (보유 종목·비중)
- 보유 종목 시세 / 관련 뉴스·시장 동향
- 그날 정리 노트 / 퀴즈·정답률 (쌓이는 학습 기록)
