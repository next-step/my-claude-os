# 데이터 소스 (Data Sources)

> 데일리 리포트가 매일 조회하는 신뢰 가능한 증권/분석 사이트.
> AI는 이 사이트들을 웹 조회해 시세·뉴스·지표를 가져온다.

---

## 한국 주식

| 용도 | 사이트 |
|------|--------|
| 공식 시세·공시 데이터 | KRX 정보데이터시스템 (data.krx.co.kr) |
| 종목 시세·뉴스·재무·수급 | 네이버페이 증권 (finance.naver.com) |
| 전자공시 (DART) | dart.fss.or.kr |
| 애널리스트 컨센서스 | 한경 컨센서스 (consensus.hankyung.com), FnGuide |
| 기업 분석 | 증권사 리서치 리포트 (각 증권사 HTS/홈페이지) |

## 사용 원칙
- **단일 출처 맹신 금지.** 시세는 1차(증권사), 해석은 교차 확인.
- 커뮤니티(종목토론방·트위터)는 **참고만**, 매매 단독 근거로 쓰지 않는다.
- 데일리 리포트 하단에 실제 조회한 출처를 남긴다.
