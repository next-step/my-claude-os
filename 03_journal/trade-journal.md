# 매매 일지 (Trade Journal)

> 워크플로우 4단계 "결과 학습"의 **유일한 진짜 입력값.**
> 숫자(손익)보다 **그때 무슨 생각/감정이었는지**가 핵심이다. 매매 직후 바로 적는다(기억 왜곡 전).
> 새 기록은 맨 위에 추가(최신순).

---

## 기록 템플릿 (복붙해서 사용)

```
### [YYYY-MM-DD] 종목명(티커) — 매수/매도 | 버킷: 적립식/단타

- 가격 / 수량 / 금액:
- (단타) 손절가 / 목표가:
- 진입·청산 근거:
- 그때 감정 (1~5, FOMO/공포/확신/충동 등):
- 원칙 준수 여부: 손절선 지켰나 / 베팅한도 지켰나 / 근거 있었나
- 결과 (청산 시 손익 %·금액):
- 복기 한 줄 (다음에 뭘 다르게?):
```
