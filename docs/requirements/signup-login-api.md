# 요구사항 — 회원가입·로그인 API (FastAPI)

> 이 파일은 **사람이 직접 채웁니다.** 채운 내용을 바탕으로 /prd 가 PRD를 만듭니다.
> 모르는 항목은 비워두어도 됩니다(PRD 단계에서 "확인이 필요한 점"으로 남깁니다).

## 한 줄 요약
(이 기능을 한 문장으로 — 예: "이메일/비밀번호로 회원가입하고 로그인해 토큰을 받는 REST API")
이메일/비밀번호로 회원가입하고 로그인하는 RestAPI

## 배경 / 문제
아직 뭘 할지 구체적이지 않음.

## 원하는 동작
(엔드포인트별로 — 가능하면 시나리오로)
- 회원가입: 회원가입의 필수값 아이디/패스워드 이름 끝
- 로그인: 아이디 패스워드로 로그인합니다.
- (선택) 내 정보 조회 / 로그아웃 / 토큰 갱신: 이건 지금 안할래

## 제약 / 조건
(꼭 지켜야 할 것)
- 인증 방식 (예: JWT / 세션): JWT 을 이용합니다.
- 비밀번호 저장 (예: bcrypt 해싱): sha512 를 사용하여 비밀번호를 저장합니다.
- DB (예: PostgreSQL / SQLite): SQLITe
- 파이썬/FastAPI 버전: python 3.1.2, FastAPI 0.111.1
- 기타(이메일 인증, 속도 제한 등): 이메일 인증 없고 속도 제한도 없음

## 이번에 안 할 것 (범위 밖)
(예: 소셜 로그인, 비밀번호 찾기, 관리자 기능 등 — 이번엔 제외할 것)
소셜로그인, 비밀번호 찾기 등 원하는 동작을 제외한 나머지 기능은 안함
