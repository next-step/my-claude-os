# XP & 레벨 시스템 화면 명세

> 작성일: 2026-06-26
> 연관 PRD: habit-tracker/docs/planning/prd-xp-level.md
> 연관 디자인 시스템: habit-tracker/docs/design/design-system.md

---

## 목적

루틴 체크라는 단발성 행동을 "내가 성장하고 있다"는 장기적 서사로 연결하여, 사용자가 내일도 앱을 열게 만드는 시각적 동기를 제공한다.

---

## 사용자 플로우

### 플로우 A — 일반 홈 진입 (XP 바 표시)

```
홈 진입
  → XP 바 컴포넌트 렌더링
  → 프로그레스 바 0% → 실제값으로 스르륵 채워짐 (0.4s ease-out)
  → 사용자, 현재 레벨과 남은 XP 확인
```

### 플로우 B — 루틴 체크 후 레벨업 발생

```
루틴 체크 완료 (POST /check/{id})
  → 서버: XP 적립 → 레벨업 감지 → 플래시 속성 세팅
  → 홈으로 redirect (GET /)
  → 레벨업 배너 최상단 슬라이드 인 (0.3s)
  → XP 바: 새 레벨 기준 0%에서 현재 진행률로 채워짐
  → 5초 후 배너 자동 페이드 아웃 (0.5s)
  → [또는] 사용자가 ✕ 클릭 시 즉시 제거
```

### 플로우 C — 만렙(Lv.7 전설) 상태

```
홈 진입 (totalXp >= 15000)
  → XP 바 자리에 만렙 카드 렌더링
  → 프로그레스 바 없음 / "★ 최고 레벨 달성 ★" 텍스트 표시
  → 골드 shimmer 애니메이션 반복 재생
```

## 레이아웃

### 화면 1 — 홈 화면 전체 구조

```
┌──────────────────────────────────────────┐
│  네비게이션 바 (오늘 / 루틴 / 통계 / 기록) │
├──────────────────────────────────────────┤
│  ╔══════════════════════════════════════╗ │  ← 레벨업 배너 (조건부)
│  ║  🌱 → 🌿  묘목에 도달했어요!    [✕] ║ │     위에서 슬라이드 인
│  ╚══════════════════════════════════════╝ │
├──────────────────────────────────────────┤
│  ┌──────────────────────────────────────┐ │
│  │  🌱  새싹  Lv.1                      │ │  ← XP 바 컴포넌트
│  │  ████████░░░░░░░░░░  80 / 200 XP    │ │     루틴 목록 위, 상단
│  └──────────────────────────────────────┘ │
│                                          │
│  오늘의 루틴            2026년 6월 26일   │
│  ─────────────────────────────────────   │
│  [루틴 카드]                              │
│  [루틴 카드]                              │
│  [루틴 카드]                              │
└──────────────────────────────────────────┘
```

**배치 결정 이유:**
XP 바를 루틴 목록보다 위에 놓는 것은 "내가 왜 이 루틴을 하는가"를 화면에 진입하는 순간 상기시키기 위해서다. 루틴 카드가 먼저 보이면 XP는 부가 정보가 된다. 순서를 바꾸면 XP가 맥락이 된다.

### 화면 2 — XP 바 컴포넌트 (일반 상태 Lv.1~6)

```
┌──────────────────────────────────────────────┐
│                                              │  border-left: 4px, 레벨별 색상
│  [이모지 2rem]  [레벨명]  Lv.[N]             │  ← 상단 행
│                                              │
│  ┌──────────────────────────────────────┐   │
│  │ ████████████░░░░░░░░░░░░░░░░░░░░░░  │   │  ← 프로그레스 트랙 (8px height)
│  └──────────────────────────────────────┘   │  fill: 그린 그라데이션
│                                              │
│  [현재레벨내XP] / [다음레벨까지XP] XP        │  ← 보조 텍스트, 우측 정렬
└──────────────────────────────────────────────┘
```

- 카드 패딩: 16px (--space-md)
- 카드 모서리: 12px (rounded-3)
- 이모지와 레벨명 사이 간격: 8px (--space-sm)
- 프로그레스 바와 텍스트 행 사이 간격: 8px

### 화면 3 — XP 바 컴포넌트 (만렙 상태 Lv.7 전설)

```
╔══════════════════════════════════════════════╗  ← border: 2px solid 골드
║                                              ║     background: 골드 그라데이션 틴트
║  [⚡ 2.5rem]  전설  Lv.7                    ║
║                                              ║
║            ★ 최고 레벨 달성 ★               ║  ← 중앙 정렬, 골드 볼드
║                                              ║     shimmer 애니메이션 적용
╚══════════════════════════════════════════════╝
```

- 프로그레스 바 없음 (숨김 처리, `th:unless="${userLevel.name == 'LEGEND'}"`)
- XP 수치 텍스트 없음
- 배경: `linear-gradient(135deg, #FFFBEB, #FEF3C7)`
- 테두리: `2px solid #F59E0B`
- "★ 최고 레벨 달성 ★" 텍스트: `color: #B45309`, `font-weight: 700`, `font-size: 15px`
- ⚡ 이모지: 2.5rem (다른 레벨 2rem 대비 25% 크게)

**설계 의도:**
만렙은 목표가 사라진 상태다. "더 이상 채울 게 없다"를 빈 바로 표현하면 공허하다.
대신 골드 카드 전체가 "달성의 기념비"처럼 기능하도록 설계했다.

### 화면 4 — 레벨업 축하 배너

```
┌──────────────────────────────────────────────────────────┐
│  [이전이모지] → [새이모지]  [새레벨명]에 도달했어요!   [✕] │
└──────────────────────────────────────────────────────────┘
```

- 위치: 컨테이너 최상단, 네비게이션 바 바로 아래
- 높이: 고정값 없음, 텍스트 기준 최소 56px (터치 타겟 확보)
- 좌우 패딩: 16px (--space-md)
- 배경: `linear-gradient(90deg, #F59E0B, #FB923C)` (골드 → 오렌지)
- 텍스트 색상: `#FFFFFF`, `font-weight: 600`
- 닫기 버튼(✕): 우측 상단, 최소 44×44px 터치 타겟, `color: rgba(255,255,255,0.85)`
- 이모지 크기: 1.5rem (인라인 텍스트 기준)
- 화살표(→): 텍스트 문자, 이모지 사이 --space-sm 여백

## 컴포넌트 상태

### XP 바 컴포넌트 (xp-bar)

| 상태 | 조건 | 프로그레스 바 | 수치 텍스트 | 배경/테두리 |
|------|------|------------|------------|-----------|
| 기본 (Lv.1–6) | `userLevel != LEGEND` | 표시, 그린 그라데이션 fill | `{xpInLevel} / {xpToNext} XP` | 흰 배경, 레벨색 좌측 보더 |
| 만렙 (Lv.7) | `userLevel == LEGEND` | 숨김 | 숨김 | 골드 배경 틴트, 골드 테두리 |
| 신규 사용자 | `totalXp == 0` | 표시, 0% | `0 / 200 XP` | 기본과 동일 |

### 레벨별 XP 바 좌측 보더 색상

| 레벨 | 이름 | 이모지 | 좌측 보더 색상 |
|------|------|--------|-------------|
| 1 | 새싹 | 🌱 | `#86EFAC` (연초록) |
| 2 | 묘목 | 🌿 | `#4ADE80` (라임) |
| 3 | 나무 | 🌳 | `#22C55E` (--color-success) |
| 4 | 숲 | 🌲 | `#16A34A` (중간 초록) |
| 5 | 정원사 | 👨‍🌾 | `#15803D` (짙은 초록) |
| 6 | 자연인 | 🏕️ | `#14532D` (딥 그린) |
| 7 | 전설 | ⚡ | `#F59E0B` (골드, 전체 테두리) |

**설계 의도:**
레벨이 올라갈수록 초록이 짙어지는 방식으로 "성장 깊이"를 색상에 녹였다.
사용자가 의식하지 않아도 "내 바 색깔이 달라졌네"를 통해 진척감을 느낄 수 있다.

### 레벨업 배너 (level-up-banner)

| 상태 | 조건 | 동작 |
|------|------|------|
| 등장 | `leveledUp == true` | 위에서 슬라이드 인 (0.3s ease-out) |
| 유지 | 등장 후 4.5s 동안 | 정지 |
| 페이드 아웃 | 4.5s 경과 후 | opacity 0으로 (0.5s ease-in) |
| 즉시 제거 | 사용자 ✕ 클릭 | 즉시 opacity 0 → DOM 제거 |
| 미표시 | `leveledUp == false` | Thymeleaf `th:if` 로 렌더 안 함 |

## 텍스트 가이드

### XP 바 컴포넌트

- 레벨 표시 형식: `[이모지] [레벨명] Lv.[N]`
  - 예시: `🌱 새싹 Lv.1`, `🌳 나무 Lv.3`, `⚡ 전설 Lv.7`
- XP 수치 형식: `[현재레벨내XP] / [다음레벨까지필요XP] XP`
  - 예시: `80 / 200 XP`, `450 / 900 XP`
  - 세 자리 이상은 쉼표 없이 표시 (1500, 7000 등 — 서버단 포매팅 불필요)
- 만렙 텍스트: `★ 최고 레벨 달성 ★`

### 레벨업 배너

- 본문 형식: `[이전이모지] → [새이모지]  [새레벨명]에 도달했어요!`
  - 예시: `🌱 → 🌿  묘목에 도달했어요!`
  - 예시: `🌲 → 👨‍🌾  정원사에 도달했어요!`
  - 화살표 앞뒤 공백 포함 (→ 가독성)
  - "에 도달했어요!" 톤: 당근 스타일 — 앱이 직접 축하를 건네는 느낌
- 닫기 버튼: `✕` (유니코드 문자, Bootstrap Icons `bi-x-lg` 대안 사용 가능)

### 빈 상태 (XP 바 단독은 빈 상태 없음)

XP 바는 항상 Lv.1 / 0 XP 상태로 시작하므로 별도 빈 상태 없음.
신규 사용자 = "0 / 200 XP" + 0% 프로그레스 바.

## 애니메이션 스펙

### XP 프로그레스 바 fill 애니메이션

```css
.xp-progress-fill {
  width: 0%;                         /* 초기값 */
  transition: width 0.4s ease-out;   /* 스르륵 채워짐 */
}
```

```js
// DOM 로드 후 실제 값으로 업데이트 (JS 필수)
document.addEventListener('DOMContentLoaded', () => {
  const fill = document.querySelector('.xp-progress-fill');
  if (fill) {
    fill.style.width = fill.dataset.percent + '%';
  }
});
```

- `data-percent` 속성: Thymeleaf로 `th:attr="data-percent=${xpProgressPercent}"` 주입
- 이유: CSS transition은 초기값(0%)에서 변경될 때만 발동. 서버 렌더링으로 바로 width를 세팅하면 애니메이션이 작동하지 않는다.

### 레벨업 배너 슬라이드 인 / 페이드 아웃

```css
.level-up-banner {
  transform: translateY(-100%);
  opacity: 1;
  transition: transform 0.3s ease-out, opacity 0.5s ease-in;
}
.level-up-banner.visible {
  transform: translateY(0);
}
.level-up-banner.fading {
  opacity: 0;
}
```

```js
const banner = document.querySelector('.level-up-banner');
if (banner) {
  // 슬라이드 인
  requestAnimationFrame(() => banner.classList.add('visible'));

  // 4.5s 후 페이드 아웃 시작 (총 5s에 완료)
  const fadeTimer = setTimeout(() => {
    banner.classList.add('fading');
    banner.addEventListener('transitionend', () => banner.remove(), { once: true });
  }, 4500);

  // 닫기 버튼
  banner.querySelector('.banner-close').addEventListener('click', () => {
    clearTimeout(fadeTimer);
    banner.remove();
  });
}
```

## 반응형 규칙

### 모바일 (< 576px)

- XP 바 컴포넌트: 풀 너비 (`width: 100%`), 좌우 여백 없음 또는 container 패딩만
- 레벨 이모지: 2rem 유지 (축소 금지 — 레벨 정체성의 핵심)
- XP 수치 텍스트: 13px (--보조 타이포), 줄바꿈 없이 한 줄 유지
  - 글자가 잘릴 경우: `overflow: hidden; text-overflow: ellipsis` 금지 — 대신 숫자 자체가 충분히 짧으므로 문제 없음
- 레벨업 배너: 풀 너비, 세로 높이 최소 56px, 텍스트 한 줄 유지 (이모지 + 텍스트)
  - 매우 긴 레벨명(예: "정원사에 도달했어요!")이 잘릴 경우: 줄바꿈 허용 (2줄까지)
- 닫기 버튼 터치 타겟: 44×44px 최소, 우측 상단 고정

### 데스크탑 (> 768px)

- XP 바 컴포넌트: container max-width 720px 안에서 전체 너비 사용
- 레벨업 배너: container 너비 (720px) 기준, 화면 전체 너비 대신 container 너비로 제한
  - 이유: 화면 전체를 덮는 것은 브리프에서 명시적으로 제한. 넓은 화면에서 container 밖으로 뻗어나가면 "무시할 수 없는 크기"가 아닌 "방해 요소"가 된다.

## 구현 클래스 목록 (프론트 참조)

| CSS 클래스 | 역할 |
|-----------|------|
| `.xp-bar-container` | XP 바 전체 감싸는 카드 컨테이너 |
| `.xp-bar-header` | 이모지 + 레벨명 + Lv.N 행 |
| `.xp-level-emoji` | 레벨 이모지 (font-size: 2rem) |
| `.xp-progress-track` | 프로그레스 바 배경 트랙 |
| `.xp-progress-fill` | 실제 fill (width: JS 제어) |
| `.xp-text` | "N / M XP" 수치 텍스트 |
| `.xp-bar-legend` | 만렙 전용 shimmer 카드 |
| `.xp-legend-text` | "★ 최고 레벨 달성 ★" 텍스트 |
| `.level-up-banner` | 레벨업 축하 배너 전체 |
| `.banner-close` | 배너 닫기 버튼 |

Thymeleaf 분기 포인트:
- `th:if="${userLevel.name != 'LEGEND'}"` → 프로그레스 바 + XP 수치 표시
- `th:if="${userLevel.name == 'LEGEND'}"` → 만렙 텍스트 + shimmer 표시
- `th:if="${leveledUp}"` → 레벨업 배너 렌더링
